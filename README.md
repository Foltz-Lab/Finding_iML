# Finding_eML

Code repository for the paper:

> **Memory-like NK cells naturally occur in healthy individuals and are enriched within solid tumors**

This repository contains classifier scripts run on pan-cancer cohorts and documentation for identifying and analyzing memory-like (ML) NK cells using machine learning model.
